# Twilight

Twilight is a free, open-source Discord AI chatbot built to feel fast, capable, and approachable, powered by **Google Gemma 4** on **Google AI Studio**, built on the [Dopamine Framework](https://framework.dopaminestudios.in/). No strings attached.

A **Dopamine Studios** Project.

## Invite Twilight

Add Twilight to your server or start a DM from the [official website](https://twilight.dopaminestudios.in/).

Questions, suggestions, or issues? Join the support server [here](https://discord.gg/xT4RVAADeU).

## How to talk to Twilight

| Where | How                                                        |
|:------|:-----------------------------------------------------------|
| **DMs** | Send a message directly, no mention needed.                |
| **Servers** | `@Twilight` in a message, or reply to one of its messages. |

It reads reply context, so quoting a message gives her useful background. Attach images or files when you want it to look at something. It supports common image and document types (not audio or video).

---

## Features

### AI Chat

* **Gemma 4 26B:** Responses run through Google’s Gemma 4 family via Google AI Studio, with deeper reasoning for complex prompts.

* **Streaming replies:** Answers update live as they are generated, so you are not left staring at a blank message.

* **Thinking indicators:** For harder questions, status messages reflect what it is working on (logic, code, writing, and more).

* **Attachments:** Send images or files in the same message as your prompt; Twilight analyses them in context.

* **Session memory:** It remembers the current conversation while you keep chatting. History is not stored permanently and resets after about an hour of inactivity.

* **Reply-aware context:** Replying to its messages (or quoting others) gives it the quoted text automatically.

### Commands

* `/clear dm` — Clear conversation history in a DM.

* `/clear server` — Clear conversation history for the server (admin only).

* `/ping` — Latency, uptime, and bot diagnostics.

* `/latency graph` — 24-hour API latency graph.
